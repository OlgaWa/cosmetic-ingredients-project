Cosmetic Ingredients Finder

EU CosIng Database

CosIng is the European Commission database for information on cosmetic substances and ingredients. Licence: Attribution 4.0 (CC BY 4.0).

It includes almost 30.000 substances. My dataset was obtained from the official site of the EU CosIng Database (links below). For my project I've cleaned the data and put it in MySQL database.

Here you can find the original database and learn more about it:
